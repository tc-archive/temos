# iTerm Guide

----------------------------------------------------------------------------------------------------
##### Tabs

```
1.  New Tab             : <cmd>t
2.  Change Tab          : <cmd+shift>[ (left), <cmd+shift>] (right) 
```

----------------------------------------------------------------------------------------------------
##### Splits

```
1.  New Split           : <cmd>d (vertical), <cmd+shift>d (horizontal)
2.  Change Split        : <cmd>[ (previous), <cmd>] (next) 
```
